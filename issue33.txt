The five commonly used HTML elements are:

1.<div> - Division Element:
Purpose: The <div> element is a generic container that groups together other HTML elements. It is often used to structure and organize content into sections or blocks.

Example:
<div>
    <p>This is a paragraph inside a div.</p>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>
</div>

2.<a> - Anchor Element:
Purpose: The <a> element is used to create hyperlinks, allowing users to navigate to other web pages or resources. It can also be used for in-page navigation.

Example:
<a href="https://www.example.com">Visit Example.com</a>

3.<img> - Image Element:
Purpose: The <img> element is used to embed images in a web page. It allows you to display graphics, photos, or other visual content.

Example:
<img src="image.jpg" alt="A descriptive text for the image">

4.<p> - Paragraph Element:
Purpose:The <p> element represents a paragraph of text. It is commonly used to structure and format text content on a webpage.

Example:
<p>This is my first paragraph.</p>

5.<ul> / <ol> / <li> - Lists Elements:
Purpose:<ul> (Unordered List) and <ol> (Ordered List) are used to create lists, and <li> (List Item) is used to define individual items in a list.
<ul> is typically used for bulleted lists, <ol> for numbered lists, and <li> for each list item.

Example:
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
</ol>
