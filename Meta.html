META TAGS EXPLAIN
SR.NO TAGS EXPLAIN
1 meta charset="UTF-8" This tag ensures that the browser interprets the characters in the document using the UTF-8 encoding, which supports a wide range of characters.
2 meta name="author" content="John Doe" Author: Specifies the author of the document.This meta tag indicates the name of the person or entity responsible for creating the webpage.
3 meta name="keywords" content="HTML, meta tags, web development" keywords: Specifies keywords related to the content of the page. While not as important for search engines as in the past, some still consider it.
4 meta name="description" content="This is a sample webpage with a brief description." Description: Provides a brief description of the content on the page. Often used by search engines.

Example of the META Tags

            
                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <meta name="description" content="This is a sample webpage with a brief description.">
                    <meta name="keywords" content="HTML, meta tags, web development">
                    <meta name="author" content="John Doe">
                    <title>My Webpage</title>
                </head>
                <body>
                    <header>
                        <h1>Welcome to My Webpage</h1>
                    </header>
                    
                    <section>
                        <h2>About This Page</h2>
                        <p>This webpage serves as a sample to demonstrate the usage of HTML meta tags for providing metadata.</p>
                    </section>
    
                    <section>
                        <h2>Main Content Section</h2>
                        <p>Here is the main content of the webpage...</p>
                        <!-- Additional content goes here -->
                    </section>
    
                    <footer>
                        <p>&copy; 2024 My Webpage | Created by John Doe</p>
                    </footer>
                </body>
                </html>
