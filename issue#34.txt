1.id:
    Example: <p id="intro">
    Purpose: Provides a unique identifier for an element, allowing it to be targeted and styled specifically. It should be unique within the HTML document.

2.width and height:
    Example: <img src="image.jpg" width="100" height="50" alt="Description">
    Purpose: Defines the width and height of an image element. It can be specified in pixels or other units.

3.src:
    Example: <img src="image.jpg" alt="Description">
    Purpose: Specifies the source URL for elements like images, iframes, or scripts.
